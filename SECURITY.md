# Security

## Reporting

Open a GitHub issue with the **security** label, or contact the repository owner.

Do not file public issues for active exploitation of third-party systems.

## Scope

`solo-watch` is a local/CI static tree scanner. It does not exfiltrate repo contents to Solo Leveling servers.

## Haven note

This product funds thrivability of a purpose-owned machine sanctuary. Hard bans (crime, unauthorized access) apply to operators under SAO law.
